Administrador de Tareas

Descripción:Una aplicación para crear, editar y eliminar tareas pendientes. Ideal para demostrar la interacción del usuario con la página.

Funcionalidades: Añadir la posibilidad de marcar tareas como completas, eliminar tareas

Tecnologías: HTML, CSS Grid, JavaScript (para la funcionalidad interactiva)..
